void coloreado.colorNode ( int  node,
int  maxColors 
) [inline]

Verifica si es posible colorear el nodo indicado con el primer color de la lista. Si no es posible pasa al siguiente. Al encontrar una opción válida hace una llamada recursiva con el siguiente nodo. Si no es posible colorear i+1, se le asigna el siguiente color de la lista a i. El bucle acaba cuando o bien se han coloreado todos los nodos o se han agotado las posibles combinaciones de colores (sin haber alcanzado el nodo final).
